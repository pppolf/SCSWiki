# 资料分享

这里整理同学们公开贡献的课程学习资料，包括 Markdown 笔记、PDF、Word 文档等。资料仅供学习参考，不代表课程官方要求；课程安排、实验要求和考核方式请以任课教师当学期说明为准。

## 收录边界

- 可以收录：学生原创笔记、知识点梳理、复习清单、实验环境说明、公开资源索引。
- 暂不收录：未授权教材扫描件、教师课件原文件、未公开试卷、含个人信息的作业或报告、内部系统链接。
- 文件命名建议使用英文小写短横线，例如 `chapter-01-overview.pdf`。

## 贡献方式

附件按课程放在 `docs/public/materials/课程英文名/` 目录下，资料清单统一维护在 `docs/.vitepress/materials.ts`。每门课程按“知识点笔记、实验资料、试题资料”等分类整理；试题资料中的试卷和答案应放在同一个资料组里。
