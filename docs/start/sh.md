## 其他时间

- 断电（熄灯）时间：**门禁时间往后推 $30$ 分钟，即冬季作息周一断电（熄灯）时间为 $23:00$。**
- 热水供应时间：$06:00\sim 08:30$、$12:00\sim 14:00$、$16:00\sim 23:30$。
- 洗衣机使用时间：晚上 $23:00$ 后无法使用，但是在之前已经启动的话会把衣服洗完。
- 吹风机使用时间：晚上 $23:00$ 后无法使用。
- 早晚自习时间（大一强制）
  - 早自习：周一至周五 $07:15\sim 07:45$，固定教室点名。
  - 晚自习：周一至周五（无课） $19:00\sim 21:00$（冬）/ $19:30\sim 21:30$（夏）
